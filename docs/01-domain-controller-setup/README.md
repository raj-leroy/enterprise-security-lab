# Domain Controller Setup (DC1)

## Overview
This section documents the installation and configuration of the first domain controller (DC1) for the CORP.LOCAL domain.

## Steps Completed
- Installed Windows Server 2022
- Configured static IP: 192.168.50.10
- Installed AD DS role
- Promoted server to domain controller
- Verified DNS operation

## Commands Used
dcdiag /test:dns  
nslookup corp.local  
ipconfig /all
